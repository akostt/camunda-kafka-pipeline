# Статусы строк ДО — справочник

Статусы строк ДО (позиций заказа) используются в ERP-системе **Галактика** для отслеживания
жизненного цикла отдельной позиции в заказе.

> **Формула ID:** сырой ID из базы имеет суффикс `h` и первую цифру `0`–`7`.
> Прибавить `8` к первому hex-символу, убрать `h` в конце.
> Примеры: `0001…h` → `8001…`, `4000…h` → `C000…`

> **Код** — числовой классификатор стадии строки (поле из Галактики, диапазон 01–09).

Статусы используются в поле `specification.statusId` схемы `all_order_v2` — это статус
отдельной строки (позиции) заказа, в отличие от `statusId` на уровне всего заказа
(см. [statuses.md](statuses.md)). Отмеченные **★** — выделены жёлтым в источнике и
используются в авросхеме.
