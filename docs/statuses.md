# Статусы заказа — справочник

Статусы используются в ERP-системе **Галактика** для отслеживания жизненного цикла заказа.

> **Формула ID:** сырой ID из базы имеет суффикс `h` и первую цифру `0`–`7`.
> Прибавить `8` к первому hex-символу, убрать `h` в конце.
> Примеры: `0001…h` → `8001…`, `4000…h` → `C000…`

---

## Статусы в Avro-схеме

Эти статусы используются в поле `statusId` в схеме `all_order_v2` (выделены в источнике жёлтым).
